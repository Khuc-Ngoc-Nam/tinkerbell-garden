# Triển khai UC1.1
## Bước 1: Tạo cấu trúc gốc và Docker Compose
Tại thư mục ngoài cùng `tinkerbell-garden/`, bạn tạo file `docker-compose.yml`. File này sẽ khởi chạy cả Frontend và Backend cùng lúc.

```
version: '3.8'

services:
  server:
    build: ./server
    container_name: tinkerbell_server
    ports:
      - "5000:5000"
    volumes:
      - ./server:/app
      - /app/node_modules
    environment:
      - PORT=5000
    command: npm run dev

  client:
    build: ./client
    container_name: tinkerbell_client
    ports:
      - "5173:5173"
    volumes:
      - ./client:/app
      - /app/node_modules
    command: npm run dev
```

## Bước 2: Setup Backend (Khởi tạo & Code UC1.1)
1. Mở terminal, vào thư mục server, khởi tạo `Node.js`

```
cd server
npm init -y
npm install express cors
npm install --save-dev nodemon
```

2. Tạo `Dockerfile` trong server:

```
FROM node:18-alpine
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
EXPOSE 5000
```

3. Update `server/package.json`, thêm vào script:

```
"scripts": {
  "dev": "nodemon src/app.js"
}
```

4. Bắt đầu code UC1.1 tại `server/src/modules/portal/portal.controller.js`. Do chưa có DB, ta sẽ thiết lập một Object tĩnh:

```
// server/src/modules/portal/portal.controller.js
const getParkInfo = (req, res) => {
    // Fake data thay cho Database
    const parkInfo = {
        name: "TinkerBell Garden",
        description: "Khu vui chơi trẻ em hàng đầu với nhiều tiện ích.",
        ticketPrices: [
            { type: "Vé 2 tiếng", price: 100000 },
            { type: "Vé Full ngày", price: 200000 }
        ],
        facilities: ["Khu nhà bóng", "Cầu trượt", "Workshop làm bánh"]
    };
    
    res.status(200).json({ success: true, data: parkInfo });
};

module.exports = { getParkInfo };
```

5. Tạo Route tại `server/src/modules/portal/portal.routes.js`:

```
const express = require('express');
const router = express.Router();
const { getParkInfo } = require('./portal.controller');

// GET /api/portal/info
router.get('/info', getParkInfo);

module.exports = router;
```

6. Móc nối tất cả vào `server/src/app.js`:

```
const express = require('express');
const cors = require('cors');
const portalRoutes = require('./modules/portal/portal.routes');

const app = express();

app.use(cors());
app.use(express.json());

// Đăng ký routes cho Module 1
app.use('/api/portal', portalRoutes);

const PORT = process.env.PORT || 5000;
app.listen(PORT, () => {
    console.log(`Server is running on port ${PORT}`);
});
```

## Bước 3: Setup Frontend
1. Mở terminal mới, tạo project React bằng Vite trong thư mục `client/`

```
npm create vite@latest client -- --template react
cd client
npm install axios
```

2. Tạo file `client/Dockerfile`

```FROM node:18-alpine
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
EXPOSE 5173
```

3. Sửa file `client/vite.config.js` để cho phép Docker map port ra ngoài:

```
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'

export default defineConfig({
  plugins: [react()],
  server: {
    host: true, // Thêm dòng này để Docker bắt được port
    port: 5173
  }
})
```

4. Gọi API tại `client/src/App.jsx` (Đại diện cho `HomePage.jsx` của UC1.1):

```
import { useEffect, useState } from 'react'
import axios from 'axios'

function App() {
  const [parkInfo, setParkInfo] = useState(null);

  useEffect(() => {
    // Gọi API từ Backend
    axios.get('http://localhost:5000/api/portal/info')
      .then(response => {
        setParkInfo(response.data.data);
      })
      .catch(error => console.error("Lỗi gọi API:", error));
  }, []);

  if (!parkInfo) return <div>Đang tải thông tin...</div>;

  return (
    <div style={{ padding: '20px' }}>
      <h1>{parkInfo.name}</h1>
      <p>{parkInfo.description}</p>
      
      <h2>Bảng giá:</h2>
      <ul>
        {parkInfo.ticketPrices.map((ticket, index) => (
          <li key={index}>{ticket.type}: {ticket.price.toLocaleString()} VNĐ</li>
        ))}
      </ul>

      <h2>Các khu vực vui chơi:</h2>
      <ul>
        {parkInfo.facilities.map((facility, index) => (
          <li key={index}>{facility}</li>
        ))}
      </ul>
    </div>
  )
}

export default App
```


2. Backend: Cập nhật Route

Mở file `server/src/modules/portal/portal.routes.js` để gắn đường dẫn cho API mới:

```
// server/src/modules/portal/portal.routes.js
const express = require('express');
const router = express.Router();
const { getParkInfo, getEvents } = require('./portal.controller');

// GET /api/portal/info (UC1.1)
router.get('/info', getParkInfo);

// GET /api/portal/events (UC1.2) - Thêm mới
router.get('/events', getEvents);

module.exports = router;
```

3. Frontend: Tạo Component hiển thị Sự kiện

Theo đúng cấu trúc thư mục của bạn, hãy tạo các thư mục `modules/portal/pages/` bên trong `client/src/`. Sau đó tạo file `EventsPage.jsx`:
```
// client/src/modules/portal/pages/EventsPage.jsx
import { useEffect, useState } from 'react';
import axios from 'axios';

const EventsPage = () => {
  const [events, setEvents] = useState([]);

  useEffect(() => {
    // Gọi API lấy danh sách sự kiện từ Backend
    axios.get('http://localhost:5000/api/portal/events')
      .then(response => {
        setEvents(response.data.data);
      })
      .catch(error => console.error("Lỗi gọi API sự kiện:", error));
  }, []);

  return (
    <div style={{ padding: '20px', borderTop: '2px dashed #ccc', marginTop: '20px' }}>
      <h2>Danh sách Sự kiện (UC1.2)</h2>
      {events.length === 0 ? <p>Đang tải sự kiện...</p> : (
        <div style={{ display: 'flex', gap: '20px', flexWrap: 'wrap' }}>
          {events.map(event => (
            <div key={event.id} style={{ border: '1px solid #333', padding: '15px', borderRadius: '8px', maxWidth: '300px' }}>
              <h3 style={{ margin: '0 0 10px 0', color: '#0056b3' }}>{event.name}</h3>
              <p>{event.description}</p>
              <p><strong>Từ:</strong> {event.startDate} <strong>đến</strong> {event.endDate}</p>
              <p style={{ color: 'red', fontWeight: 'bold' }}>🔥 Giảm {event.discountRate}% Early Bird</p>
              <button style={{ padding: '8px 12px', cursor: 'pointer' }}>Đăng ký ngay</button>
            </div>
          ))}
        </div>
      )}
    </div>
  );
};

export default EventsPage;
```
